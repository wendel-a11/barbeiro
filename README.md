Descrição

Projeto front-end de um site chamado "barbeiro" (HTML, CSS e JavaScript).
Provavelmente um trabalho/mini‑projeto estático com pastas como TRABALHO e assets front-end.
Tecnologias

HTML
CSS
JavaScript
Instalação e execução local Opções rápidas para ver o projeto localmente:

Abrir diretamente
Abra o arquivo index.html no navegador (se o projeto for apenas arquivos estáticos).
Usar um servidor estático (recomendado para evitar problemas com caminhos relativos)
Com Python 3: na pasta do projeto execute: python -m http.server 8000 e acesse http://localhost:8000 no navegador.
Com Node (npx serve): na pasta do projeto execute: npx serve .
Usar sua ferramenta de desenvolvimento preferida (Live Server do VS Code, etc).
Estrutura sugerida de pastas

TRABALHO/ — (conteúdo do trabalho)
index.html — página inicial (se existir)
css/ — estilos
js/ — scripts
assets/ ou img/ — imagens e recursos
